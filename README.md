# eos-virtual-pet
Interactive virtual pet built with Python and Pygame, featuring emotions, hunger, autonomous movement and user interactions.  🩷

PT - BR 
Pequeno projeto desenvolvido em Python utilizando Pygame.

## Funcionalidades

- Sistema de emoções
- Sistema de fome
- Interação com o mouse
- Objetos aleatórios no ambiente
- Movimento autônomo
- Evento de colapso ao receber muitos cliques


## Tecnologias

- Python
- Pygame

## Objetivo

Projeto criado para praticar lógica de programação, eventos, estados e desenvolvimento de aplicações interativas.
